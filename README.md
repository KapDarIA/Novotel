# Novotel — сайт бронирования номеров

Веб-приложение для бронирования номеров отеля с авторизацией, ролями и базой данных (SQLite или MySQL).

## Возможности

- **Гость** — просмотр номеров, поиск, фильтр по типу, сортировка по цене (без входа)
- **Пользователь** — регистрация, вход, бронирование, профиль с историей бронирований
- **Администратор** — добавление, редактирование и удаление номеров и услуг; экспорт бронирований за месяц в Excel
- **Услуги** — фитнес, SPA, бар, ресторан, уборка, еда в номер (только при брони номера, на даты проживания)
- **Отзывы** — оценка отеля 1–5 звёзд и текст (только для гостей с бронированием номера)
- Проверка пересечения дат бронирования для одного номера
- Хранение в БД: логин, хеш пароля, номера (название, тип, цена, картинка), бронирования

## Запуск в VS Code

1. Установите [Node.js](https://nodejs.org/) **22.5 или новее** (у вас подойдёт текущая v24).
2. Откройте папку проекта в VS Code.
3. В терминале выполните:

```bash
npm install
npm start
```

4. Откройте в браузере: **http://localhost:3000**

## Учётные записи по умолчанию

| Роль  | Логин | Пароль   |
|-------|-------|----------|
| Админ | admin | admin123 |
| Пользователь | user | user123 |

Новых пользователей можно зарегистрировать на странице «Регистрация».

## Подключение MySQL

1. Установите MySQL (или XAMPP / OpenServer с MySQL).
2. Создайте базу — выполните скрипт `schema-mysql.sql` в MySQL Workbench или:

```bash
mysql -u root -p < schema-mysql.sql
```

3. Скопируйте настройки:

```bash
copy .env.example .env
```

4. В файле `.env` укажите:

```env
DB_TYPE=mysql
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=ваш_пароль
MYSQL_DATABASE=novotel_db
```

5. Запустите сайт: `npm install` → `npm start`.

Таблицы создадутся автоматически, если их ещё нет. Демо-пользователи и номера добавятся при первом запуске.

**Без MySQL** — ничего настраивать не нужно: по умолчанию используется SQLite (`novotel.db` в папке проекта).

## Структура

- `server.js` — API и сервер
- `db/` — подключение SQLite или MySQL
- `schema-mysql.sql` — скрипт создания базы в MySQL
- `.env` — настройки подключения (создайте из `.env.example`)
- `novotel.db` — SQLite-база (если не используете MySQL)
- `public/` — HTML, CSS, JS

## Роли

| Роль | Описание |
|------|----------|
| guest | Не авторизован — иконка пользователя ведёт на вход |
| user | Зарегистрированный — профиль и бронирование |
| admin | Управление номерами + все права user |
